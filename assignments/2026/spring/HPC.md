# HPC 题目

## 1. SGEMM

GEMM（General Matrix Multiply，通用矩阵乘法），作为线性代数中的基本操作几乎无处不在。而BLAS (Basic Linear Algebra Subprograms) 基础线性代数子程序提供了一套通用的线性代数接口

| types   | name | options       | size  | arguments                    | description                   | equation       | flops  | data       |
| ------- | ---- | ------------- | ----- | ---------------------------- | ----------------------------- | -------------- | ------ | ---------- |
| s,d,c,z | gemm | transA,transB | m,n,k | alpha,A,ldA,B,ldB,beta,C,ldC | general matrix-matrixmultiply | $C=αA^∗B^∗+βC$ | $2mnk$ | $mk+nk+mn$ |

本题要求针对单精度模式 (fp32，sgemm) 进行优化，调用接口和BLAS保持一致。本题不限制CPU的架构和规模，从树莓派到AMD EPYC 9755，x86，Arm亦或是RISC-V均可以使用。

### 负载

| 场景 | A 是否转置 | B 是否转置 | M    | N    | K     |

| 2    |     否     |     否     | 8192 | 8192 | 8192  |

当然你也可以自行额外选择一些corner case并进行优化，但上述三个负载为必做部分。
